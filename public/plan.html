Nordic Studio
AI Agent Platform

One app to download, set up, and manage your own AI agent. Powered by OpenClaw — no terminal, no config files, no fuss.

🚀 One-click deploy
🤖 OpenClaw gateway
🌐 Browser relay
🗄 Custom data sources
💬 AI chat built-in
What it does
1️⃣
Download & open
User downloads the macOS app — no setup needed to open it
2️⃣
Guided setup wizard
App checks requirements, installs OpenClaw, sets API key, starts the gateway — all automatically
3️⃣
Dashboard + Chat
Full management view — status, sessions, data sources — plus the AI chat interface
4️⃣
Ongoing management
Add data sources, switch AI models, monitor gateway health, update from one place

Open Questions
Things to decide before implementation starts
❓
Q1 — App name
Is "Nordic Studio" still the right name now that it's a full setup + management platform, not just an AI chat? Options: Nordic Studio (keep), Claw (simpler), Arboretum (already in the UI), something else?
❓
Q2 — Other AI agents besides OpenClaw?
You mentioned "OpenClaw or any AI agent". Should the setup wizard support other agents (LangChain, AutoGen, CrewAI)? Or is OpenClaw-only fine for v1?
❓
Q3 — Distribution
How will users download the app? GitHub Releases, a landing page, or Mac App Store? This affects code signing and notarisation requirements.
❓
Q4 — Antigravity / IDE integration
You mentioned wanting to use Antigravity. Should the app support opening the project in an IDE directly from the UI? Or is that out of scope for now?
